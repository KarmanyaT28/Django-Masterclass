Introduction to Django

Django is a high-level Python web framework that enables developers to build robust,scalable, and secure web applications quickly. It emphasises rapid development, clean and pragmatic design, and comes with many built-in tools that eliminate the need to reinvent the wheel



1. DJANGO IS A HIGH-LEVEL PYTHON WEB FRAMEWORK

High level -> Neet not worry about low-level details of web development.
Enables us to focus on building the business logic

Python Framework -> Written in Python & used for Python based development


2. DJANGO ENABLES DEVELOPERS TO BUILD ROBUST, SCALABLE, AND SECURE APPLICATIONS QUICKLY

Robust Apps -> Apps which do not crash, manages errors gracefully

Scalable Apps -> Can handle high traffic websites without affecting performance

Secure Apps -> Provides protection against common vulnerabilities such as SQL injection, Cross-Site Scripting(XSS), Cross-Site Request Forgery(CSRF) & Clickjacking


3. DJANGO EMPHASISES RAPID DEVELOPMENT

Enables developers to build web applications quickly

Automatically sets up the database schema, handle authentication & builds the admin interface

Comes with an ORM(Object Relational Mapper), does not require us to write SQL Queries

Built in development server


4. DJANGO EMPHASISES CLEAN AND PRAGMATIC DESIGN

Every django app follows MVT (Model-View-Template) architecture

MVT architecture promotes clean and pragmatic design


5. DJANGO HAS BUILT-IN TOOLS

Authentication

URL Routing

Forms & validation

Database migration
