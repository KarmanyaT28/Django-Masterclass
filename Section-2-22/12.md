Step 1 — Go to your Django project
cd /path/to/your/project


Make sure manage.py is there:

ls

Step 2 — Open Django shell
python manage.py shell

Step 3 — Find the superuser

Paste this exactly:

from django.contrib.auth import get_user_model
User = get_user_model()

User.objects.filter(is_superuser=True).values("id", "username", "email")


You will see something like:

<QuerySet [{'id': 1, 'username': 'admin', 'email': 'admin@example.com'}]>


Note the username.

Step 4 — Reset password

Replace admin with the username you saw:

u = User.objects.get(username="admin")
u.set_password("NewStrongPassword123")
u.save()


You should see no errors.

Step 5 — Exit
exit()
